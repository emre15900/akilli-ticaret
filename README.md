# Akıllı Ticaret

Next.js 14 App Router, Apollo Client ve Redux Toolkit ile geliştirilen ürün listeleme uygulaması.

## Kurulum

1. Bağımlılıkları yükle:

```bash
yarn install
```

2. Geliştirme sunucusunu başlat:

```bash
yarn dev
```

## Test senaryoları

- `utils/barcodeMatching` için barcode-görsel eşleştirme testleri
- `store/favoritesSlice` aksiyon ve hydration senaryoları

## Gerekli minimum Node.js sürümü

```bash
>= 20.19.0
```
